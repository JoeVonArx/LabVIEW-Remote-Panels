# LabVIEW-Remote-Panels
Example Client and Server Code for Implementing a Simple cRIO GUI. 

Project save version is LabVIEW 2019.

This simple project provides example code and notes for how to use Remote Panel Server on the cRIO to allow clients on PCs on the same the LAN to download the Front Panels of VIs running on the cRIO and interact with the running cRIO code as if it were running on the client machine.
